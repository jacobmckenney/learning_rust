# Data Structures

## Linked List

-   Singly-Linked List
-   Time Complexities:
    -   Push: O(1)
    -   Pop: O(1)
    -   Contains: O(n)

## BST

-   Time Complexities:
    -   Insert: O(n)
    -   Find: O(n)
    -   Contains: O(n)

## Hash Map

-   Linked Hash Map with Rehashing
    -   Load Factor(Items/Buckets): 0.75
-   Time Complexities (Amortized):
    -   Put: O(1)
    -   Get: O(1)
    -   Remove: O(1)
    -   Contains: O(1)
    -   Resize: O(n), n = # of buckets
